## 使い方

以下のコマンドで記号コードに自動変換

```node index.js -c {変換するファイルのパス} -o {出力先ファイルパス}```


## 記号プログラミングとは？

アルファベットを用いずに、記号だけで書くプログラミングのこと。JavaScriptの場合、以下のようなメリットとデメリットがある。

- メリット
  - キーボードのアルファベットが全部が打てなくなってもプログラミングが可能

- デメリット
  - 可読性が劇的に下がる
  - パフォーマンスが下がる
  - ファイルサイズが大きくなる
